
# 高性能 Mysql

## performance_schema

- 开启Performance Schema库

> 在配置文件中添加performance_schema=on

| 表名              | 说明                             |
| ----------------- | -------------------------------- |
| setup_instruments | 设置需要开启对哪些项监控进行统计 |
| setup_consumers   | 控制是否将监控到的结果进行记录   |
|                   |                                  |
